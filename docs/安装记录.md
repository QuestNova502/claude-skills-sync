### 三、deanpeters/product-manager-prompts（参考库）

**重要说明**: 这不是传统的Claude Skills，而是**产品经理AI提示词模板集合**。

#### 📚 包含的内容

**Prompts目录**（主要提示词）:
- `jobs-to-be-done.md` - JTBD框架分析
- `value-proposition-canvas.md` - 价值主张画布
- `user-story-template.md` - 用户故事模板
- `competitive-analysis.md` - 竞品分析框架
- `product-roadmap.md` - 产品路线图规划
- 等等...

**Flows目录**（工作流）:
- 完整的产品管理工作流程
- 从发现到交付的端到端流程

**Prompt Generators**（提示词生成器）:
- 自动生成定制化的产品管理提示词

**Storytelling**（叙事技巧）:
- 产品故事讲述模板
- IKEA式指导步骤生成器

#### 💡 如何使用

这些不是自动加载的skills，而是**复制粘贴使用的提示词模板**：

1. **浏览模板**:
   ```bash
   cat ~/.claude/skills/product-manager-prompts/prompts/jobs-to-be-done.md
   ```

2. **复制提示词到对话**:
   - 打开感兴趣的.md文件
   - 复制提示词内容
   - 粘贴到Claude对话中
   - 按照引导回答问题

3. **示例使用场景**:
   ```
   你：（粘贴jobs-to-be-done.md的提示词）
   Claude：好的，让我们用JTBD框架分析你的产品...
   （开始引导式对话）
   ```

#### 🎯 适合你的场景

作为产品经理，这些提示词非常适合：
- ✅ 用户研究（JTBD分析）
- ✅ 竞品分析（结构化框架）
- ✅ 产品战略（价值主张设计）
- ✅ 需求管理（用户故事编写）
- ✅ 路线图规划

**推荐开始使用的提示词**:
1. `jobs-to-be-done.md` - 了解用户需求
2. `competitive-analysis.md` - 竞品分析（你的强项）
3. `user-story-template.md` - 写更好的用户故事
4. `value-proposition-canvas.md` - 设计产品价值主张

